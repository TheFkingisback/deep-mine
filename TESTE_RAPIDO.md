# Teste Rápido - Deep Mine

## Como Testar o Jogo:

1. **Abra no navegador:** http://localhost:3003/
2. **Abra o Console** (pressione F12, aba Console)

### Teste 1: Jogo Carrega?
- [ ] Você vê blocos marrons (terra)?
- [ ] Você vê um quadrado azul (jogador)?
- [ ] Você vê o HUD no topo (ouro, profundidade, inventário)?
- [ ] Você vê botões no rodapé?

### Teste 2: Cliques Funcionam?
- [ ] Clique em blocos ao redor do jogador
- [ ] Os blocos desaparecem quando clicados?
- [ ] Itens caem (círculos coloridos)?
- [ ] O jogador cai quando remove blocos abaixo?

### Teste 3: Coletar Itens
- [ ] Clique nos itens que caíram
- [ ] O contador de inventário aumenta?

### Teste 4: Ir para Superfície
- [ ] Clique no botão "⬆ Surface" (canto inferior esquerdo)
- [ ] A cena muda para mostrar céu azul e grama?
- [ ] Você vê o jogador na superfície?
- [ ] Há 3 áreas interativas (entrada da mina, venda, loja)?

### Teste 5: Voltar para Mina
- [ ] Na superfície, clique no botão "⬇ DESCEND"
- [ ] Selecione uma profundidade
- [ ] Você volta para a cena de mineração?
- [ ] Os blocos aparecem novamente?

## Se algo não funciona:

**Copie a mensagem de erro do console e me envie!**

O console mostrará algo como:
```
❌ Error: ...
```

## Controles do Jogo:

- **Mouse/Clique**: Cavar blocos, interagir, coletar itens
- **NÃO use teclado** - o jogo não tem controles de teclado ainda!
